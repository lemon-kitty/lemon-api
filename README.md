## 项目介绍

这是一个提供API接口的开放平台。



## 项目选型

### 前端

* React
* Ant Design Pro

- Umi UI

* Ant Design & Procomponents
* OpenAPI

### 后端

* Spring Boot 2

- Spring MVC

* MySQL

* Spring Cloud Gateway

* API签名认证

* Dubbo

* MyBatis-Plus

* MyBatis X
